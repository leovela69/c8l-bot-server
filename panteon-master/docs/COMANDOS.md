# 📋 Comandos de Telegram — PANTEÓN MASTER

## Comandos del Sistema (Zeus)

| Comando | Descripción |
|---------|------------|
| `/estado` | Muestra el estado completo del sistema |
| `/ayuda` | Lista todos los comandos disponibles |
| `/paneles` | Muestra los bots activos y su estado |

## Comandos de Creación Musical (Vulcano + Apolo)

| Comando | Descripción |
|---------|------------|
| `/musica [tema]` | Genera letra de canción Bolero-House |
| `/producir [tema]` | Produce un track completo (letra + melodía + estructura) |
| `/remix [cancion]` | Genera un remix de una canción existente |
| `/beat [estilo]` | Genera un beat/instrumental |
| `/letra [tema]` | Solo genera la letra sin producción |
